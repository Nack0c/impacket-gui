# 使用方式
1.下载impacket
```
git https://github.com/fortra/impacket.git

pip install -r requirements.txt
```

2.下载本项目
```
git https://github.com/yutianqaq/impacket-gui.git
```

3.安装 Pyqt 等依赖

4.运行
